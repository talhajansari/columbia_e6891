Tipos de pacotes de Sound Ruler

     Atualmente há 4 tipos de pacotes de SoundRuler disponíveis. Verões auto-executáveis são mais rápidas do que versões em script, mas possuem menos opções de edição.

soundruler-x.x.x.linux.rpm (compilado em RedHat 9.0)
soundruler-x.x.x.linux.tar.gz
      Auto-executável para MS Windows. Não necessita de Matlab para rodar. O pacote tar.gz contém o código fonte de instalação genérica para Linux.

soundruler-x.x.x.linux.scripts.tar.gz
     M-files de script para Linux. Requer Matlab 6.x para rodar. Este pacote contém os arquivos fonte do código para Linux.

soundruler-x.x.x.windows.zip
     Auto-executável para MS Windows. Não necessita de Matlab para rodar.

soundruler-x.x.x.windows.scripts.zip
      M-files de script para MS Windows. Requer Matlab 6.x para rodar. Este pacote contém os arquivos fonte do código para Windows.

 

Requerimentos de sistema

     O Sound Ruler por si só não necessita de muitos recursos, mas análises acústicas exigem muito do processador. Quanto mais extensivas e detalhadas forem as análises, maiores serão as vantagens de um computador mais rápido. Embora o Sound Ruler rode bem em um Pentium 2 de 233 MHz, mesmo a atualização de um Pentium 4 de 1.7 GHz para um Pentium 4 de 2.4 GHz pode reduzir o tempo de análise significativamente.

Versões auto-executáveis
     Pentium 100 (486s não foram testados)
     32 MB de memória RAM
     50 MB de espaço livre no disco rígido

Versões de scripts
      Matlab versão 6.1 ou posterior
      5 MB de espaço livre no disco rígido

 

Instalação / Desinstalação

Windows auto-executável

Instalação:
1- Descomprima o arquivo soundruler_windows_0.9xx.zip para um diretório temporário (será necessário um programa descompressor como Winzip).
2- Execute setup.exe (ou setup) e siga as instruções na tela.

Desinstalação:
1- Vá em Iniciar-Configurações-Painel de Controle-Adicionar/Remover Programas.
2- Selecione SoundRuler na lista de programas.
3- Se arquivos houverem side salvos nos dire'torios do SoundRuler estes diretórios não serão removidos durante a desinstalação.
4- Uma chamada para o diretório de instalação do SoundRuler pode ter permanecido no arquivo C:\autoexec.bat.

Linux auto-executável

Para soundruler-x.x.x.linux.rpm
 Instalação:
1- su
2- cd folder_where_soundruler-x.x.x.linux.rpm_is
3- rpm -Uvh soundruler-x.x.x.linux.rpm

Desinstalação:
1- su
2- rpm -e soundruler

Para soundruler-x.x.x.linux.tar.gz
Instalação:
1- su
2- tar -xvzf /path_where_it_is/soundruler-x.x.x.linux.tar.gz.
3- cd soundruler-x.x.x.
4- vi Makefile (para verificar se os caminhos estão certos para o seu sistema)
5- make install
6- digite soundruler ou sruler para rodar, como qualquer usuário

Desinstalação:
1- Delete o diretório do SoundRuler (/usr/share/soundruler).
2- Delete soundruler e sruler de /usr/bin.
3- Delete /usr/share/man/man1/soundruler.1.gz.
4- Delete /usr/share/man/pt_BR/man1/soundruler-pt_BR.1.gz.
5- Delete a linha export LD_LIBRARY_PATH=/matlablibspath/bin/glnx86 do arquivo /home/yourusername/.bash_profile or /etc/profile.

NOTA: O arquivo spec para geração de RPMs está em soundruler-x.x.x.linux.tar.gz.

M-files de scripts

Instalação automática (Windows apenas)
1- descomprima o SoundRuler para um diretório
2- abra o Matlab
3- entre "cd nome_de_caminho", onde nome_de_caminho é o local onde os arquivos do SoundRuler foram colocados. Ex: cd 'c:\program files\soundruler'
4- entre Setupsruler
Quando terminar, entre sruler na tela do Matlab para rodar o SoundRuler

Instalação manual
1- descomprima o SoundRuler para um diretório
2- abra o Matlab
3- adicione o caminho para o diretório onde o SoundRuler foi colocado ao "path" do Matlab, ou mude para o diretório do SoundRuler. Ex:cd 'c:\program files\soundruler'
4- execute soundruler.m

O que a instalação automática faz? Esta é uma instalação limpa. Ela apenas copia sruler.m para o diretório matlab/work. O diretório /work está no caminho do Matlab por padrão. Quando sruler é digitado, o script adiciona o diretório de instalação do SoundRuler ao caminho do Matlab e executa o soundruler.

Desinstalação:
1- Faça cópias de segurança de arquivos que tenha sido salvos no diretório de instalação do SoundRuler.
2- Delete o diretório de instalação do SoundRuler.
